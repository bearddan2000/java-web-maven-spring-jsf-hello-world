# java-web-maven-spring-jsf-hello-world

## Description
A POC for springboot web app with jsf primefaces support.

## Tech stack
- java
- maven
  - springboot
  - jsf
  - bootstrap

## Docker stack
- openjdk:11-jdk

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
